# map-marker

[![Greenkeeper badge](https://badges.greenkeeper.io/mapmeld/street-web-scrape.svg)](https://greenkeeper.io/)

Select one of the roads in my geolocation (or a clicked area) to mark with a blocker ball.

* jQuery
* Leaflet.js
* OpenStreetMap
